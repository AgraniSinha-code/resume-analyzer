# Resume Analyzer

This is a simple project I built using Python and Flask.

It analyzes a resume (PDF) and shows:
- Resume score
- Skills present in the resume
- Missing skills
- Suggestions for improvement

I created this project to practice web development and understand how real-world tools analyze resumes.

## Tech Used
- Python (Flask)
- HTML, CSS
- PyPDF2

## How to Run
1. Install required libraries:
pip install flask PyPDF2

2. Run the project:
python app.py

3. Open in browser:
http://127.0.0.1:5000/
