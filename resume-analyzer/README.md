# 🚀 AI Resume Analyzer

A web-based application that analyzes resumes and provides insights such as resume score, detected skills, missing skills, and improvement suggestions.

## 🔍 Features
- Upload PDF resume
- Extract and analyze text from resume
- Detect technical skills
- Calculate resume score
- Identify missing skills
- Provide improvement suggestions
- Job role matching with percentage

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS (Bootstrap)
- PyPDF2

## ▶️ How to Run

1. Clone the repository:
[GitHub Repo](https://github.com/AgraniSinha-code/resume-analyzer.git)

2. Install dependencies:
pip install flask PyPDF2

3. Run the app:
python app.py

4. Open in browser:
http://127.0.0.1:5000/
